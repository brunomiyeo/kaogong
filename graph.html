📐 图形推理

一、命题形式

形式观察方法规律寻找
一组图逐项推：①→②→③→④→⑤→?相邻比较找规律
一组图(间隔)①→③→⑤，②→④→?奇偶分开观察
两组图第一组①→②→③找规律，第二组④→⑤→?用规律
九宫格横行/竖行/米字/Z型中间为?考虑米字
分组分类先找共性再找差异第一眼看不出去看选项

二、解题思维框架

元素特征方向具体内容
组成相同位置规律平移 / 旋转 / 翻转
组成相似样式规律遍历 / 加减同异 / 黑白运算
组成不同属性规律对称性 / 曲直性 / 开闭性
组成不同数量规律点 / 线 / 角 / 面 / 素

拆分思维

位置拆分：上下/内外/左右分开找规律

样式拆分：横竖/黑白/曲直分开看

对比思维

对比选项（看不变） / 对比题干（看相邻）

三、十五字口诀

定性：屈臣氏整风
①屈——曲直性 ②臣——对称性 ③氏——相同相似图形 ④整——整体部分 ⑤风——封闭开放

什么时候考虑？
· 图形比较随意、生活化（阴影、简笔画等）
· 图形比较凌乱、有不协调图形

定量：点线面角素
什么时候考虑？
· 图形越来越复杂，数量在增加，点线很多
· 图形规整且复杂

四、位置规律

平移

维度类型
方向直线(上下/左右/斜对角线) / 绕圈(顺/逆时针)
步数固定 / 递增减(等差/等比)
路径从头(循环跑) / 调头(反弹跑)

整体看 vs 分开看

  • 两个或多个黑块相连 → 看作一组
  • 黑块间隔白块数量恒定 → 看作一组
  • 就近原则：最短移动距离为同一元素
  • 位置重合：个别图黑块减少

宫格型

中间不变 → 优先顺逆时针走外圈

中间变化且有规律 → 内外圈分开看

行数量不变 → 优先左右移动；列数量不变 → 优先上下移动

五、样式规律

遍历：每种元素在每个位置都出现一次

加减同异：相同部分保留/去掉

黑白运算：黑+白=? 按规则运算

六、属性规律

对称性

考查点方法
性质轴对称/中心对称/轴+中心对称
对称轴数量恒定/交替/递增减/运算(加减乘除)
对称轴方向横/竖/斜/依次变化(顺逆时针旋转)
对称轴+原图经过的点/线/面数量/与原图线的关系

开闭性

开放(有缺口) / 封闭(整体封闭) / 半开半闭

曲直性

属性情况
性质全直/全曲/曲+直
数量直线数/曲线数各自呈规律
位置上曲下直/内曲外直等

七、笔画数（重点！）

特征图：五角星、圆相切/相交、「日」字变形、「田」字变形、出头端点

一笔画：奇点数=0或2 且 连在一起的图形

多笔画：笔画数 = 奇点数 ÷ 2

奇点：一个点引出的射线数为奇数

龙飞一笔画：去除封闭空间后，剩余几笔就是几笔画

八、面

同时出现数面、数线、数笔画、数点的特征图时，优先数面
  • 所有面：分开数三角形/四边形/多边形个数
  • 三角形可重复数！
  • 单个面：最大面的形状/面积/属性

九、点/角/素

点

大树叉、线条交叉明显、切点多、圆相交时考虑

类型：圆点/交点/切点/出头点

角

口诀：折线折出多个角，留了缺口别忘角；横平竖直多垂直，直角数量来试试

只数小于180°的角；直线与直线相交才叫角

素

特征：很多小元素 / 生活化图形 / 黑色粗线条

考查：总数/个别种类数量/种类间运算/元素间换算

十、空间类

六面体展开图

相对面：同行/同列隔一面 / Z两端 → 不能同时出现
公共边：①构成直角的两条边 ②一列连着4个面，最两侧两条边 ③3+1的L型同方向两条边

三板斧：①对立面(13/24/56不同时出现) ②14直接平移四格不变 ③56平移两格(中心对称直接平移)

截面图

几何体能截出不能截出
六面体三角形(锐角)/四边形/五边形/六边形正五边形/直角梯形
圆柱体圆/椭圆/矩形/抛物线梯形

三视图

长对正、高平齐、宽相等。曲面平滑交接无线。原图有线就有线。

立体拼合

分层画图法：化立体为平面、化拼为拆

十一、特殊题型

黑白块

  • 对称性（整体/黑/白部分对称）
  • 面（黑面积=白面积）
  • 连接（部分数/笔画数/连接方式）
  • 相邻比较（按行按列对比）

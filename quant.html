🔢 数量关系

目标：15题 · 10-15分钟 · 60%正确率（随缘模块）

一、鸡兔同笼

假设都做对或都赚到
做错数量 = (理想情况 - 现实情况) ÷ 单个差值

例：获胜5分/打平2分/失败-1分，20场积61分3场平
理想：17场全胜=85分 → 失败=(85-55)/(5-(-1))=5 → 胜12场，胜率60%

二、浓度问题

公式：浓度 = 溶质 ÷ 溶液
溶质不变法：无论溶液如何改变，溶质不改变

十字相乘法：
A浓度、B浓度 → 混合R → (R-B)/(A-R) = A质量/B质量

凡是 A=B/C 形式的比例关系，均可看成类浓度问题
利用十字相乘求出的比例关系一定是 C(分母) 之比

三、牛吃草

核心：原有草量 = (牛的吃速 - 草的生长速度) × 时间

四、容斥问题

两集合：A∪B = A + B - A∩B
三集合：A∪B∪C = A+B+C - A∩B - A∩C - B∩C + A∩B∩C

①占比和超过100% → 一定有交集
②问至多 → 考虑「包含」，至多有「较小比重」
③问至少 → 考虑「相斥」，a+b-100% 或 a+b-总量

五、植树问题

类型公式
两端不植树总长 = 间隔 × (棵数 + 1)
一端/环形总长 = 间隔 × 棵数
两端植树总长 = 间隔 × (棵数 - 1)

六、行程问题

路程 = 速度 × 时间
追及：路程差 = 速度差 × 时间
相遇：路程和 = 速度和 × 时间
流水：顺速 = 船速 + 水速；逆速 = 船速 - 水速

七、工程问题

工作总量 = 效率 × 时间
设工作总量为1，效率为每天完成的比例
多人合作：总效率 = 各人效率之和

八、利润问题

利润 = 售价 - 成本
利润率 = 利润 / 成本
折扣价 = 定价 × 折扣率

九、排列组合与概率

排列：P(n,m) = n! / (n-m)!（有序）
组合：C(n,m) = n! / (m!(n-m)!)（无序）
概率 = 满足条件的情况数 / 总情况数

十、钟表问题

分针速度：360°/60min = 6°/min
时针速度：30°/60min = 0.5°/min
分针追时针：5.5°/min

十一、最值问题

至少…保证…：最坏情况 + 1
至多…不超过…：求最大值约束
N个数之和固定，求最大值：其余N-1个尽可能小

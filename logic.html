一、归因论证（重点！）

题型特征

既成事实 + 原因分析 → 结论

有时原因分析省略，直接给出结论或建议

例：学校门口堵车（既成事实）→ 开车接送孩子（原因）

归因 vs 因果

归因因果
条件要有既成事实（完成时态）不需要既成事实
例子小明考上了，是因为他的努力休息不好会导致考试发挥不好

对比实验归因

题干特征：有完整对比实验过程，并根据结果给出原因分析

质疑方式说明
另有他因除了实验中的变量，还有其他因素导致了差异
因果倒置不是A导致B，而是B导致A
否定此因证明这个原因并不是真正的原因

一般归因

没有完整对比实验过程，与对比实验归因质疑、支持的做题方法完全相同！

选项特征

特例选项（有些/个别...）正确概率极低
实验瑕疵类（样本量不足、样本不具代表性）正确概率极低

二、一般质疑

归因类之外的质疑题目，即非谈论因果关系

题目形式：论据 + 结论（隐含有论证过程）

情况质疑方式
无论据有结论
(背景+结论)		质疑结论（A和非A的矛盾）
例：有人提出XX方案→质疑这个方案的可行性
有论据有结论
(大段论据+结论)		①质疑论据（极少数）
②质疑论证过程（最常见！）
③质疑结论
核心原则：主体一致，话题一致，贴合题干！这是所有语言类做题的大前提

三、支持与前提假设

解释说明

解释了为什么得到此结论（提供额外理由）

前提假设（支持类的一种）

选：没他不行（没有这个假设，论证就不成立）
不选：有他更好（有了更好，没有也可以）

断点搭桥：选包含「论据关键词 + 结论新内容」的选项

论据 → 关键词 → 【搭桥选项】 → 新内容 → 结论

支持力度比较

必要条件 > 补充论据 > 解释说明（一般情况）

四、比例类论证

一种「数学 + 逻辑」的一般质疑题

常见错误：用「分子」代替「分子/分母」得出结论

例：78%的儿童中耳炎患者来自二手烟家庭 → 还要看所有人群中二手烟家庭占比
如果二手烟家庭占80%，那说明二手烟反而可能有保护作用

秒杀：找选项带数字的分母部分，构成正确比例关系

五、推出类

逻辑基础

A → B（充分条件 → 必要条件）

必要条件标志词：基础、前提、关键、必须、需要、离不开、依赖、必不可少、不可或缺

互为充分必要条件：「A当且仅当B」= A→B 且 B→A

命题形式

命题公式说明
原命题A → B
否命题¬A → ¬B与原命题无推出关系
逆命题B → A与原命题无推出关系
逆否命题¬B → ¬A等价于原命题

等价推出

A→B→C = ¬C→¬B→¬A
忠于箭头（肯定向右，否定向左）

只找同伙，无论对错
选项以C或¬A开头 → 推不出任何东西
比较类、原因类都不是逻辑推理过程，不选！

六、真假分析（矛盾法）

一真一假

①找矛盾 → 找到存在的矛盾关系，必然一真一假
②看其余 → 结合真假数量确定矛盾以外的真假
③矛盾关系：
· A 和 ¬A
· A或B 和 ¬A且¬B
· A且B 和 ¬A或¬B
· 所有A都是B 和 有些A不是B
· 所有A都不是B 和 有些A是B
· 可能P 和 必然¬P
· A→B 和 A且¬B

两真两假

先找矛盾，再假设

结合真假数量确定矛盾以外的真假情况

七、秒杀口诀

口诀用法示例

3+2找相同，相同归3做排除

九、定义判断

解题思路：重在理解，联系实际，找到现实所指

最大坑：不要因未体现某一特点就排除，关注「被定义词」中不包含的信息

技巧

  • 问啥看啥：问「属于」还是「不属于」
  • 同构选项：意思相近结构相似 → 同时排除
  • 对比枚举型：选项一一对应
  • 专业名词：建立在道德基础上，符合「常理」
  • 涉及时间：注意选项时间点设置
  • 涉及变化：选项应有「以前」和「现在」对比

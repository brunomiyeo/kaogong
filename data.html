📊 资料分析

目标：20题 · 30分钟 · 正确率80%+（尽量全对）

一、基础概念

概念定义易混
基期A与谁相比，谁为基期
现期B当前统计周期
增长量X绝对差值（增加多少）
增长率R相对差异（增加快慢）=增速=增幅
变化幅度看增速绝对值与增长幅度不同
同比基期为历史同期
环比基期为上个统计周期
比重部分/整体，单位相同
平均总量/份数，单位不同均前每后做分母
倍数=增长率+1A是B的5倍=A比B多4倍
翻番翻一番=2倍翻n番=2^n倍
名义增长率实际增长率+通货膨胀率

二、ABRX 四量关系

A=前期 | B=本期 | X=变化量 | R=变化率

求X：R>10%→415份数法 / R<10%→假设分配法 / R<5%→X≈AR≈BR
求A：A=B/(1+R)
求R：R=X/A

隔年增长率（笑脸公式）
R = R₁+R₂+R₁×R₂ ≈ R₁+R₂

比值增长率：(R₁-R₂)/(1+R₂)
用于「人均」「单位面积」「平均」等关键词

三、比重类

本期比重 = 部分 / 整体

前期比重 = 本期比重 × (1+整体R) / (1+部分R)
= 本期比重 × (1+R₂)/(1+R₁) 反过来

比重趋势秒杀：R₁(部分) > R₂(整体) → 上升

比重差秒杀：比重差 < |R₁-R₂|
精确公式：比重差 = a/B × (R₁-R₂)

比值增长率 vs 比重差
比值增长率：R=X/A，单位不同，选项%，有「人均」「平均增长率」字样
比重差：A/B-a/b，单位相同，选项百分点，有「占…比重」字样

四、比较类

增量秒杀：B越大R越大 → X越大

双线法：R=X/A
分母变大分子变小 → 分数变小
分子长的快 → 分数变大

姚潘追及思想：
高的增量更大 → 差距拉大
矮的增量更大 → 差距缩小

增速变化表述：
增幅扩大5点 → 去年=今年-5%
降幅扩大5点 → 去年=-(|今年|-5%)

五、平均类

均前每后做分母（平均每也是每）

年均增长量 = (末期-基期) / n
年均增长率：(1+R)^n = 末期/基期
R很小且选项接近时：1+nR ≈ 1+nR

闰年：能被4整除（整百年需被400整除）
2011年初到2018年末 = 2010年末到2018年末，n=8

六、盐水类（十字交叉法）

A部分a + B部分b → 混合浓度r
A/B = (r-b)/(a-r)

适用：①A=B/C ②整体分两部分 ③求出来的比例是前期之比

例：20%和30%配成22% → 8%:2% = 4:1

七、速算技巧

技巧用法场景
化除为乘|r|≤5%: A/(1+r)≈A(1-r)求基期
415份数法R=25%→基期4份,X=1份,本期5份R>10%
假设分配法X=AR，画分配树R<10%万能
截位法除法直接舍弃无需四舍五入选项差距大
十字交叉混合→求量之比混合增长率
高位叠加加法从高位加起精确求和
削峰填谷找基准值，根据偏离求求平均
乘法拆分45%=50%-5%, 15%=10%+5%百分数乘法
尾数法精确求和差选项末位不同

八、特殊考点

拉动增长 = 部分增量 / 整体前期
贡献率 = 部分增量 / 整体增量

容斥：a+b>100%必有交集；交集最少=a+b-100%；交集是区间
